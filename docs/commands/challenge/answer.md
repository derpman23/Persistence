# Answer

**Aliases:** ans

**Description:** Set the answer for the challenge.

**Syntax:** answer [&lt;answer&gt;, set &lt;answer&gt;, role &lt;role&gt;, channel &lt;channel&gt;, reset]

---

Sets the answer to a challenge, sets the channel to alert solvers in, set the role to give solvers, and let admins reset the challenge.
